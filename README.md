# 📊 E-Commerce Sales Analytics Dashboard

An end-to-end Business Intelligence project developed in Power BI to analyze e-commerce sales performance, customer behavior, product trends, and geographical distribution.

The dashboard enables data-driven decision-making through KPI monitoring, trend analysis, customer analytics, and interactive reporting. This project demonstrates practical skills in data modeling, DAX calculations, Power Query transformations, and dashboard development.

# 📁 Dataset

The project uses e-commerce transactional data containing:

- Customer Information
- Product Details
- Sales Transactions
- Store Information
- Time-based Sales Records

The data was transformed and modeled using Power Query and a Star Schema architecture to support efficient reporting and analytics.

# 🏗 Data Model

### Fact Table

- Sales Transactions

### Dimension Tables

- Customer Dimension
- Product Dimension
- Store Dimension
- Time Dimension

This structure improves dashboard efficiency, scalability, and analytical flexibility.
